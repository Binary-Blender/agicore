# The Core Insight

Current AI workflows are highly manual.

Example:

```plaintext
Conversation
↓
Human notices useful abstraction
↓
Human manually asks:
"turn this into a skill doc"
↓
AI restructures discussion
↓
Structured artifact created
↓
Human manually moves artifact elsewhere
```

This creates:
- friction
- context duplication
- repeated prompting
- workflow interruption
- unnecessary cognitive overhead

The key realization:

> The transition itself should become a first-class orchestration primitive.
