UX SPECIFICATIONS

9.1 HIGH-LEVEL INTERACTION MODEL
The user flows within Screamo follow a triad: ingest → prioritize → respond.

Primary workflow:

User opens the dashboard.

High-priority messages appear first.

User selects a message.

Screamo displays:

Message content

Classification metadata

Priority justification

AI-generated draft

User edits or approves the draft.

Final message is sent and feedback is logged.

SPC metrics update immediately.

Secondary workflow:

Feed scanner identifies LinkedIn opportunities.

User sees suggested comments.

User approves, edits, or rejects.

If approved, the system posts via LinkedIn API.

System continuously updates the SPC controller and models across all workflows.
